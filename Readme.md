# Unity용 CsvDeserializer

## 개요

`CsvDeserializer`는 CSV 문자열 데이터를 C# 객체의 `List<T>`로 역직렬화하기 위한 Unity용 경량 유틸리티입니다. 빠르게 동작하도록 설계되었으며, 가비지 컬렉션 발생을 최소화하는 것을 목표로 합니다.

역직렬화 시 CSV 헤더 행을 기준으로, 대상 C# 클래스의 `public` 필드 또는 프로퍼티에 컬럼을 매핑합니다. 매핑은 대소문자를 구분하지 않습니다.

## 개발 의도

이 코드는 "조금 더 빠른 특수해"보다 "환경 제약이 적고 유지보수가 쉬운 범용 해법"에 더 무게를 두고 작성되었습니다.

다음과 같은 제약을 의도적으로 두었습니다.

1. 런타임 코드 생성(Expression Tree 등)을 사용하지 않습니다.
   iOS처럼 AoT(Ahead-of-Time) 컴파일이 강제되는 환경에서는 이런 방식이 제약이나 문제를 만들 수 있기 때문입니다.
2. 파싱 대상 타입에 어트리뷰트로 스키마를 강제하지 않습니다.
   약간의 성능 이득보다 데이터 구조 정의와 유지보수 비용이 더 크다고 판단했습니다.
3. Source Generator를 사용하지 않습니다.
   학습 곡선이 높고, 사용자가 생성 코드를 이해하거나 디버깅하기 어렵다고 봤습니다.

결과적으로 이 라이브러리는 순수한 `string` 파싱과 `reflection`만으로 동작하는 범위 안에서, 최대한 고속이고 가볍게 동작하는 구현을 목표로 합니다.

## 기본 사용법

1. **데이터 클래스를 정의합니다.**

   ```csharp
   public class MyData { public string Name; public int Score; /* ... 추가 필드/프로퍼티 */ }
   ```

2. **역직렬화합니다.** (`csvString`에 CSV 데이터가 들어 있다고 가정)

   ```csharp
   List<MyData> dataList = CsvDeserializer.Deserialize<MyData>(csvString);
   ```

## 주요 기능 및 세부 사항

이 유틸리티는 다음과 같은 기능을 지원합니다.

* 사용자 지정 구분자
* 복합 타입을 위한 사용자 지정 타입 변환기
* 바인딩 범위 설정(필드, 프로퍼티, 또는 둘 다)
* CSV 셀이 비어 있을 때 C# 문자열 기본값 유지 여부 설정
* **RFC-4180**을 준수하는 CSV 파싱
* 따옴표로 감싼 필드, 이스케이프 문자, 줄바꿈 처리
* 타입 불일치 및 CSV 형식 오류 처리

자세한 사용 예제, 예외 처리 방식, 컬럼 불일치 상황에서의 동작은 데모 파일의 `CsvDeserializerDemo.cs`를 참고하면 됩니다. 해당 예제를 통해 `CsvDeserializer`의 전체 동작 방식을 확인할 수 있습니다.
